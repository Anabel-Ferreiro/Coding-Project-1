# User Feedback Form
Coding Project 1- Capturing and Responding to User Input

## Description
An interactive feedback for built with HTML, CSS, and JavaScript featuring real-time character counting, tooltips, form validation, and dynamic feedback display.

## How to Run
1. Open index.html in a browser
2. Fill out the form fields
3. View real-time character count and tooltips
4. Submit to see feedback displayed dynamically
